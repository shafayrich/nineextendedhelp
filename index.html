📖 Core Dictionary & Glossary

Vertex / Vertices

The highest (maximum) or lowest (minimum) turning point of a parabola. Found exactly on the axis of symmetry.

Roots / X-Intercepts

The points where a graph crosses the x-axis. Also known as "solutions" or "zeros" because they occur where y = 0.

Y-Intercept

The specific point where the graph crosses the y-axis (where x = 0). In y = ax² + bx + c, the y-intercept is 'c'.

Domain

The complete set of all possible input values (x-values) that a function can accept without breaking mathematical rules.

Range

The complete set of all possible resulting output values (y-values) that a function produces.

Parabola

The perfectly symmetrical U-shaped or upside-down U-shaped curve created when graphing a quadratic equation.

Discriminant (Δ)

The expression b² - 4ac (found inside the square root of the quadratic formula). It reveals exactly how many roots the equation has.

Coefficient

The number explicitly multiplying a variable. In the term 5x², the coefficient is 5. If it's just x², the coefficient is an invisible 1.

GCF

Greatest Common Factor. The largest shared number or variable that divides evenly into all terms of an expression.

Indices / Exponents

A superscript number indicating how many times a base is multiplied by itself (e.g., in x³, 3 is the index).

Axis of Symmetry

The invisible vertical mirror line dividing a parabola perfectly in half. Calculated using the formula x = -b / (2a).

Asymptote

An invisible line that a graph continually approaches but never physically touches or crosses.

Binomial

An algebraic expression containing exactly two terms, usually connected by a plus or minus (e.g., x + 4).

Factorise

The reverse of expanding. Taking an expanded expression and compressing it back into multiplied brackets.

Inequality

A mathematical statement comparing two quantities that are not equal, using symbols like <, >, ≤, or ≥.

Sign Diagram

A 1D number line used to determine where a function is positive (+) or negative (-) by testing intervals between roots.
